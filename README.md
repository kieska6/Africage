Africage

# À propos

Africage est une plateforme de mise en relation P2P pour l'envoi et le transport de colis entre particuliers à travers l'Afrique. La plateforme connecte les expéditeurs avec des voyageurs vérifiés pour des livraisons rapides, économiques et sécurisées.

## Fonctionnalités

- **Publication d'annonces de colis** : Les expéditeurs peuvent publier des annonces détaillées pour leurs colis
- **Recherche de trajets** : Les voyageurs peuvent consulter les annonces correspondant à leur itinéraire
- **Système de notation** : Évaluation mutuelle entre expéditeurs et voyageurs pour renforcer la confiance
- **Messagerie intégrée** : Communication directe entre les utilisateurs via une messagerie sécurisée
- **Vérification d'identité (KYC)** : Processus de vérification pour assurer la sécurité de la communauté
- **Paiement sécurisé** : Système de tokens pour des transactions sécurisées
- **Tableau de bord administrateur** : Interface de gestion complète pour les administrateurs

## Technologies

- **Frontend** : React, TypeScript, Vite, Tailwind CSS
- **Backend** : Supabase (PostgreSQL, Auth, Storage, Realtime)
- **Paiements** : Stripe
- **Déploiement** : Vercel

## Installation

1. Cloner le dépôt
2. Installer les dépendances : `npm install`
3. Configurer les variables d'environnement (.env)
4. Lancer le développement : `npm run dev`
